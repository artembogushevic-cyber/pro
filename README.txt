ГОТОВЫЙ САЙТ «ПРОФ ПРОКОЛ» ДЛЯ GITHUB PAGES

Что внутри:
- index.html — главная страница сайта
- style.css — дизайн
- script.js — анимации
- assets — фотографии и логотип

Как загрузить на GitHub:
1. Создайте репозиторий.
2. Нажмите Add file → Upload files.
3. Загрузите НЕ АРХИВ, а файлы из этой папки:
   index.html, style.css, script.js и папку assets.
4. Нажмите Commit changes.
5. Откройте Settings → Pages.
6. Source: Deploy from a branch.
7. Branch: main, folder: /root.
8. Нажмите Save.

Через несколько минут сайт появится по ссылке GitHub Pages.
